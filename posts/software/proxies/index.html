    Forward Proxy #

    Forward proxy or proxy sits in front of a set of clients (eg. web broswers) and forwards requests to a server on the internet. Purpose is to filter content, block requests or protect identity of clients.

    Reverse Proxy #

    A Reverse proxy sits in front of web servers and forward client (eg. web broswer) requests to the servers. Purpose is to increase security & performance of web service provided by the web servers.
