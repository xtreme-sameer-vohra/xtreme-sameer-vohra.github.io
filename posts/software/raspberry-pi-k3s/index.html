    RPi with K3S with Knative

    August 27, 2021
    Software
    Tech, Kubernetes, K3S, Raspberry Pi, Knative

    The process #

    Use Raspberry Pi OS #

    Install OS #

    • Download the Raspberry Pi OS Lite OS, extract the zip file
    • Install it to the SD Card Instructions
    diskutil list` # Get disk#
diskutil unmountDisk /dev/diskN
sudo dd bs=1m if=path_of_your_image.img of=/dev/rdiskN; sync
sudo diskutil eject /dev/rdiskN

    replacing N with the disk number

    SSH #

    Connect the SD Card again

    • For SSH sudo touch /Volumes/boot/ssh
    • For Wifi link
    ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
country=<Insert 2 letter ISO 3166-1 country code here>
update_config=1

network={
 ssid="<Name of your wireless LAN>"
 psk="<Password for your wireless LAN>"
}

    into /Volumes/boot/wpa_supplicant.conf

    Boot up the RPi #

    • SSH onto the RPi ssh pi@raspberrypi.local #password is raspberry
    • Use sudo raspi-config
      • Change passwd
      • Change hostname
      • Change GPU Mem to 16
    • Setup an ssh key
      • ssh-keygen
      • ssh-copy-id -i SOME_KEY_PUB_FILE pi@raspberrypi.local

    Setup for K3S #

    • Edit /boot/cmdline.txt

      cgroup_enable=cpuset cgroup_memory=1 cgroup_enable=memory

      add to the end of the line and reboot.

    • Install K3S

      curl -sfL https://get.k3s.io | sh -

    • The KUBECONFIG file is available at /etc/rancher/k3s/k3s.yaml
